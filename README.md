# SWFighter-TS

> The Force is with this game.

A Star Wars-themed browser-based space shooter built with TypeScript and Vite. Pilot your X-Wing through wave after wave of enemy ships and asteroids, from training runs to the ultimate showdown against the SithFighter.

## Quick Start

```bash
# Install dependencies
npm install

# Run dev server with hot reload
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## How to Play

**Controls:**
- ↑ ↓ ← → — Move your X-Wing
- S — Shoot

**Objective:**
Survive the incoming waves of enemies and defeat the SithFighter to win!

## Game Features

- **Progressive Difficulty**: Start with training asteroids, then face TieFighters, Shuttles, StarDestroyers, the Deathstar, and finally the SithFighter
- **Score System**: Earn points for every hit (+10) and destroyed enemy
- **Dynamic Enemies**:
  - TieFighters with swarm tactics
  - Homing missile-wielding Shuttles
  - Capital ships (StarDestroyer, Deathstar)
  - Teleporting SithFighter boss
- **Visual Effects**: Explosions, damage pulses, tractor beams, and boss animations
- **Game Over & Victory Screens**: Epic endings for defeat and triumph

## Tech Stack

- **TypeScript** with strict mode enabled
- **Vite** for fast bundling
- **CSS animations** for visual effects
- **Factory & Assembler patterns** for clean entity creation
